# cosmetic-sales-report---python--sql-powerbidashboard
Cosmetic Shop Sales Analysis Dashboard
Project Overview

This project focuses on analyzing cosmetic shop sales data to identify customer behavior, sales performance, and business growth opportunities. The analysis was performed using Excel, Python, SQL, and Power BI to transform raw data into meaningful business insights.

**Project Workflow**
Explored and understood the raw sales data in Excel.
Cleaned and preprocessed the dataset using Python.
Used SQL queries to analyze customer and sales patterns.
Created an interactive Power BI dashboard to visualize key business metrics.
Key Insights
Identified active and inactive customers.
Analyzed total sales performance and revenue trends.
Determined top-performing products and categories.
Found areas where sales were declining or losses were occurring.
Tracked customer purchasing behavior and retention.
Highlighted opportunities to improve profitability and customer engagement.
